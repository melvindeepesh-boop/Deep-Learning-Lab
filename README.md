# 🧠 Deep Learning Lab

Welcome to my **Deep Learning Lab** repository! This repository contains practical Deep Learning experiments implemented using **TensorFlow**, **Keras**, and **Google Colab** as part of my Artificial Intelligence and Machine Learning coursework.

## 📚 About

This repository demonstrates the implementation of fundamental Deep Learning concepts, including:

- Deep Neural Networks (DNN)
- Convolutional Neural Networks (CNN)
- Recurrent Neural Networks (RNN)
- Long Short-Term Memory (LSTM)
- Autoencoders
- Generative Adversarial Networks (GAN)

Each experiment includes source code, explanations, and results.
